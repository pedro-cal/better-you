# Conventions
- Folder layout: `/app` (router), `/src/{features,components,lib,state,styles}`
- API calls go in `src/lib/api/*`
- Use Tanstack Query for fetching + caching
- Keep components small; prefer composition
- Commit style: Conventional Commits
