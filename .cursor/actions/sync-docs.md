# Sync Documentation

## Description
Update all related documentation files when adding or modifying any project documentation to maintain consistency and cross-references.

## Instructions

When the user adds or updates documentation:

1. **Analyze the new/updated file** to understand its purpose and key content
2. **Update these related files** with appropriate cross-references:
   - `README.md` - Add link in documentation section
   - `STATUS.md` - Add to "Key Files & Their Purposes" section
   - `EXTERNAL_AGENT_HANDOFF.md` - Include relevant context if needed
   - Related docs in `docs/` folder - Add cross-references where logical
   - `mobile/README.md` - Add reference if relevant to mobile development

3. **Maintain consistency** in terminology and linking patterns
4. **Ensure bidirectional links** where appropriate

Execute updates systematically while preserving existing structure and content.
