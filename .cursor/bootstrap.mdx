# Goal
Create/maintain an Expo RN app (TypeScript) with Expo Router, React Query, Zod env, MMKV storage, and Notifications.

## Musts
- Keep router in `/app`, app code in `/src`
- Use `src/lib/env.ts` for config from `app.config.ts` extra
- Provide `QueryClientProvider` in `app/_layout.tsx`
- Prefer functions over classes; no heavy inheritance
- Write unit tests for new utils/components

## Output
- New files and edits + runnable instructions
