# Agent Memory & Guidelines

## Critical Rules

### 1. Do ONLY What Is Asked
- When user asks for "1 thing", create exactly 1 thing
- Do not create additional files, documentation, or explanations without explicit permission
- Do not anticipate needs or create "helpful" extras

### 2. Ask Before Creating Multiple Files
- If you think multiple files might be helpful, ASK first
- Get explicit permission before creating more than what was requested
- User will tell you if they want additional files

### 3. Be Precise and Minimal
- Follow instructions exactly as given
- Don't add scope or features not requested
- Keep responses focused on the specific ask

## Project Context
- This is the Better You project - a mobile-first habit tracking platform
- User values efficiency and precision
- User has established documentation patterns and doesn't want unnecessary files

## Lesson Learned
- 2026-01-25: User asked for 1 Cursor action, I created 4 files. User corrected this behavior.
- Always create exactly what is requested, nothing more.
